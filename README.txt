This MATLAB FileExchange entry contains files related to the "Matching Image Features" video of the AUVSI Foundation: Computer Vision Training.

examples - Files used in the video
[~] detectFeatures_start.m : Example starter MATLAB code file to detect features in an image.
[~] detectFeatures.m : Example MATLAB code file to detect features in an image using
		       Minimum Eigen Value, BRISK and MSER algorithms.
[~] matchFrames_start.m : Example starter MATLAB code file to match features between two images and visualize them.
[~] matchFrames.m : Example MATLAB code file to match features between two images and visualize them.
[~] superTitle.m : Utility MATLAB function to insert a 'Super Title' centered over a group of subplots.
[~] sampleFrames.mat : Example MAT-file with video frames showing green buoys.
[~] sampleGrayFrames.mat : Example MAT-file with video frames showing green buoys - Grayscale version.

exercises - Files for exercise and practice tasks
[~] auvsi_ipcv_exercises_03_note.pdf : Note introducing the exercise tasks and presenting respective solutions
[~] featureDetectStart.m : Example starter MATLAB code file for the exercise "Identifying a Traffic Sign with Features"
[~] featureDetect.m : Solution MATLAB code file for the exercise "Identifying a Traffic Sign with Features"
[~] matchDetect.m : Solution MATLAB code file for the exercise "Detect Traffic Sign with Matched Features"
[~] thresholdImage.m : Helper MATLAB function which thresholds the given image to extract the red background of a 			               traffic sign
[~] stop.mat : MAT-file with a video frame showing a stop sign
[~] template.mat : MAT-file with a template stop sign image
[~] yield.mat : MAT-file with a video frame showing a yield sign
[~] yield2.mat : MAT-file with a video frame showing a yield sign
